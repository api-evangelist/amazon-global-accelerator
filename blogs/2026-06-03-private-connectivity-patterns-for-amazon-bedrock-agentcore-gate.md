This guide outlines four private connectivity patterns for routing AgentCore Gateway traffic securely: MCP server targets via VPC Lattice private endpoints, REST API targets using private endpoints, Regional REST API Gateway targets through VPC Links, and Lambda functions leveraging Hyperplane ENIs for private VPC access.
